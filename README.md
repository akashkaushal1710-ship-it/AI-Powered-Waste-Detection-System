# AI-Powered Waste Detection System

A computer vision project that detects and classifies waste materials using AI models.

## Tech Stack
- Python
- OpenCV
- YOLO
- Streamlit

## Features
- Detects plastic, paper, metal, and glass waste
- Real-time object detection
- Confidence score display

Built an AI-based waste detection system using YOLO, OpenCV, and Python for real-time classification of recyclable and non-recyclable waste.
